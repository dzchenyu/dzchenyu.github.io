html

实体引用
文本

基本结构

  • <!DOCTYPE html> ：文档声明

  • <html> ：定义整个根目录

    • lang=“en” 指定网页的语言

  • <head> ：元数据，不会直接显示

    • meta charset=”utf-8” :定义字符编码
    • meta name=”viewport” content=”width=device-width,initial scale=1.0” 优化移动端浏览体验
    • title 定义网页标题

  • <body> ：所有的可见内容

html标签

文本
  • <h1> - <h6>：表示标题，<h1> 是最高级别标题，<h6> 是最低级别标题。
  • <p>：表示段落。
  • <br>：换行标签，不需要结束标签。
  • <hr>：水平线，表示主题的分隔。
列表
  • <ul>：无序列表，用于创建带有圆点的列表项。
  • <ol>：有序列表，用于创建带编号的列表项。
  • <li>：列表项，用于在无序或有序列表中定义每一项。
链接和媒体
  • <table>：定义一个表格。
  • <tr>：定义表格中的一行。
  • <td>：定义单元格（表格数据）。
  • <th>：定义表格头部单元格（表头）。
表单
  • <form>：定义表单，用于提交用户输入。
    • **action**：指定表单数据提交的目标 URL，也就是服务器端处理表单数据的脚本地址。例如 <form action="http://example.com/submit.php">，表示表单数据会被发送到 http://example.com/submit.php 这个页面进行处理。
    • **method**：用于确定表单数据提交的方式，常见取值有 GET 和 POST。
    • **enctype**：规定在发送表单数据之前如何对其进行编码，默认值是 application/x-www-form-urlencoded，适用于大多数情况，会将表单数据编码成键值对形式；如果表单中有文件上传等操作，需要将其设置为 multipart/form-data，以便正确处理二进制数据。例如 <form enctype="multipart/form-data"> 用于包含文件上传的表单
  • <input>：定义各种类型的输入字段，比如文本框、单选按钮、复选框等。

  • type=

    • **text**：创建一个单行文本输入框，用户可以在其中输入文本内容，placeholder 属性则显示一个提示性的文本，当输入框为空时提示用户应该输入什么内容。

    • **password**：用于创建密码输入框，输入的内容会以黑点等掩码形式显示，起到保密作用，像 <input type="password" name="password" placeholder="请输入密码">。

    • **radio**：创建单选按钮，搭配多个使用，几个只能选择一个

      • <input type="radio" name="cjjj" checked>阅读
<input type="radio" name="cjjj">睡觉

html属性

通用属性
  • id：定义元素的唯一标识符
  • class：为元素指定一个或多个类名，常用于CSS或JavaScript选择器
  • style：定义元素的内联样式
特定标签属性
  • <a> 标签的属性：
    • href：指定链接的目标URL地址,也可以为文件 index/hhh
    • target：指定链接如何在浏览器中打开。值 _blank 会在新窗口或新标签页中打开链接。
    • rel：定义链接与当前文档的关系，常见的有 noopener、nofollow 等
  • <img> 标签的属性：
    • src：指定图片的URL地址。
    • alt：当图片加载失败时显示的文本，有助于搜索引擎优化和辅助技术（如屏幕阅读器）。
    • width 和 height：定义图片的显示宽度和高度，可以使用像素或百分比。
  • <input> 标签的属性：
    • type：指定输入字段的类型，比如文本（text）、密码（password）、按钮（button）等。
    • placeholder：在输入字段为空时显示的提示文本。
    • value：定义默认值。
    • name：表单提交时的字段名称。

html其他概念

注释

注释不会在浏览器中显示，主要用于给开发者添加说明或备注

html语义化
  • <header>：定义页面或区块的头部。
  • <nav>：定义导航链接的集合。
  • <main>：定义文档的主要内容区域。
  • <section>：定义一个独立的内容区块。
  • <article>：定义一个独立的内容片段，比如一篇文章或博客。
  • <footer>：定义页面或区块的底部。
块级元素与行内元素
  • 块级元素：占据一行的全部宽度，并会在前后产生换行。常见的块级元素有 <div>、<p>、<h1> 等。
  • 行内元素：仅占据其内容的宽度，不会产生换行。常见的行内元素有 <span>、<a>、<img> 等。

html5

新的语义标签
  • <article>：独立的文章或内容块。
  • <aside>：定义侧边栏内容，通常与页面的主要内容相关联。
  • <figure> 和 <figcaption>：用于包含图像、图表或表格及其标题。
媒体标签
  • <audio>：用于嵌入音频内容。
  • <video>：用于嵌入视频内容。

CSS

CSS（层叠样式表，Cascading Style Sheets）是一种用于描述HTML或XML文档外观的样式表语言。它控制页面的布局、颜色、字体等，使网页的内容和样式分离，从而提高页面的可维护性和灵活性。

基本概念

语法结构
作用方式

  • 内联样式：直接在HTML元素的 style 属性中定义样式

    • <p style="color: red;">这是红色的文本</p>

  • 外部样式表：通过外部的CSS文件引入样式，通常使用 <link> 标签将CSS文件链接到HTML文档中

    • <head>
    <link rel="stylesheet" href="styles.css">
</head>

  • 类选择器：选择具有特定类名的元素，使用 . 表示。例如，选择所有 class="intro" 的元素：

    • .intro { font-size: 20px; }

  • ID选择器：选择具有特定ID的元素，使用 # 表示。例如，选择ID为 header 的元素：

    • #header { background-color: grey; }
组合选择器

  • 后代选择器：选择某个元素内部的所有指定后代元素。例如，选择所有位于 div 元素内部的 p 元素

    • div p { color: green; }

  • 并集选择器：选择多个不同元素，使用逗号 , 分隔。例如，选择所有 h1 和 p 元素

    • h1, p { margin: 10px; }

  • :focus：选中获得焦点的元素。

    • input:focus {
  border-color: blue;
}

  • :nth-child()：选择元素的特定子元素。例如，选择列表中的第二个 li 元素

    • li:nth-child(2) { color: blue; }

  • **[attribute=value]**：选中具有指定属性和值的元素

    • [type="submit"] {
  background-color: blue;
  color: white;
}

  • **[attribute*=value]**：选中属性值包含指定值的元素。

    • [class*="button"] {
  padding: 10px;
}

  • **font-size**：设置文本的大小，单位可以是 px、em、rem、百分比等。

    • p { font-size: 16px; }

  • **text-align**：设置文本的对齐方式，可以是 left、right、center 或 justify。

    • p { text-align: center; }
颜色和背景属性

  • **background-color**：设置元素的背景颜色

    • div { background-color: lightblue; }

  • **opacity**：设置元素的不透明度，取值范围是0到1。

    • div {opacity:0.8}

  • **margin**：设置元素的外边距，用来控制元素与周围元素的距离。可以单独设置四个方向的边距。

    • p { margin: 20px; }

  • **border**：设置元素的边框，可以设置边框的宽度、样式和颜色。

    • div { border: 1px solid black; }

  • **z-index**：控制元素的堆叠顺序，数值越大，元素越靠前。（对 static 属性的不起作用）

    • div { z-index: 10; }
    1. relative

      相对定位。元素相对于其原始位置进行定位。

    2. absolute

      绝对定位。元素相对于最近的已定位（即 position 不为 static）的祖先元素进行定位。如果没有这样的祖先元素，则相对于初始包含块（通常是 html 或 body）。

    3. fixed

      固定定位。元素相对于浏览器窗口进行定位，不随页面滚动而移动。

    4. sticky

      粘性定位。元素在特定的阈值内表现为相对定位，一旦滚动超过该阈值，则表现为固定定位。

显示与隐藏属性

  • **display**：控制元素的显示方式，常见的取值有 block、inline、inline-block、none。

    • p { display: block; }
